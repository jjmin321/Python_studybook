# 👨‍🎓 Python_앞으로의 학습 목표 👨‍🎓

> [파이썬 데이터분석 및 머신러닝 학습 단계 요약 정리]

1. 파이썬 기초 : 리스트를 비롯한 파이썬 특유의 자료형 접근 방법을 익히고 전통 언어의 루프 
구조를 간결하게 구현할 수 있는 방법을 익힌다, 클래스를 구현하고 멤버와 메서드를 추가하여
모듈로 사용할 수 있도록 한다

2. 데이터 수집, 분석 및 시각화 : numpy를 사용하여 2~n차원 배열의 사용법을 익힌다
pandas 를 사용하여 데이터 프레임을 읽어와서 분석하는 방법을 익힌다
웹크롤링과 웹스크레이핑 기법을 익힌다
matplotlib를 사용하여 그래프로 출력하는 방법을 익힌다
seaborn을 사용하여 좀더 간편한 시각화 방법을 익힌다

3. 데이터 전처리 : scikit-learn을 사용하여 표준화, 정규화 처리 방법을 익힌다,.
로그변환 , 이상치와 결측치 제거 기법을 익힌다
PCA(주성분분석)를 사용한 차원의 축소 및 군집화(clustering)등의 비지도 학습 방법을 익힌다

4. 머신러닝 구현 : scikit-learn을 사용하여 DecisionTree와 RandomForest등 의 분류기 등을
익힌다, 앙상블 모델, K-Fold 교차검증, 부스팅 알고리즘(XGBoost, LightGBM) 등을 익힌다

5. 딥러닝(신경망) 구현 : 구글 텐서플로나 케라스를 사용하여 신경망 구현 방법을 익힌다
Linear Regression, Logistic Regression, Classification 기법등을 익히고
CNN, RNN, DQN 등 다양한 신경망 구현 방법을 익힌다
영상이미지 인식기법을 익힌다

6. 자연언어처리 : nltk, konlpy 등을 사용하여 형태소 분석 및 토큰화와 벡터화 방법을 익히고
텍스트분석,감성분석, 챗봇구현등 자연언어처리 기법을 익힌다
BERT 와 같은 공개된 우수한 pre-training 모델을 사용하여 정확도를 올린다

7. kaggle등 국내외 다양한 데이터 과학 경진대회에 참여하여 경험을 쌓는다
